拉格朗日乘法和KKT条件

千秋摄于黄山

最近看了 《最优化理论与方法》, 学习了常见的优化问题以及解决方法。同时，之前学习 SVM 时，对拉格朗日乘法以及 KKT 条件有一些疑问，这里做一下学习笔记，便于后面复习和理解！

Notion: 同时参考了网上找到的一个PPT（见附件），觉得写的相当棒！


主要设计四种优化问题：

  1. 无约束优化
  2. 等式约束优化
  3. 不等式约束优化
  4. 等式和不等式约束优化


讨论的对象是优化问题的标准形式:


$$ \begin{aligned} &\min_{x\in X } \quad f(x) \\ &s.t \quad \begin{cases} g_i(x) = 0 \quad i = 1,2...\\ h_j(x) <= 0 \quad j = 1,2...\\ \end{cases} \end{aligned} $$
公式（1）：优化问题的标准形式

1. 无约束优化


无约束优化的标准形式为：

$\begin{aligned}
&\min_{x\in X } \quad f(x) \
\end{aligned}$
公式（2）：无约束优化的标准形式


假设f(x)在X上处处可导，则局部最优点 $x^{\star}$的充分必要条件为：
$\bigtriangledown_{x} f(x^{\star}) = 0 \quad and \quad G = \bigtriangledown^2_{x}f(^{\star}) >= 0$。
公式（3）

证明


将f(x)使用泰勒公式在$x^{\star}$处展开：
$\begin{aligned}
&f(x) = f(x^{\star}) + \bigtriangledown^Tf(x^{\star})(x-x^{\star}) + \frac{1}{2}(x-x^{\star})^T\bigtriangledown^2f(x^{\star})(x-x^{\star}) + O(||x-x^{\star}||) \
\end{aligned}$
公式（4）
​

充分条件

$\begin{aligned}
&f(x) - f(x^{\star}) = \frac{1}{2}(x-x^{\star})^T\bigtriangledown^2f(x^{\star})(x-x^{\star}) \geq 0 \quad\to充分条件 \
\end{aligned}$
​

必要条件

1. 若f(x) 在 $x^{\star}$ 处可导且取得局部最小值，则有$\bigtriangledown f(x^{\star}) = 0$

证明，使用反证法：
​

$\begin{aligned}
&假设 f(x^{\star}) \neq 0 \
&f(x) = f(x^{\star}) + \bigtriangledown ^Tf(x^{\star})(x-x^{\star}) \
& 令 (x-x^{\star}) = -\bigtriangledown f(x^{\star}) \
&f(x) - f(x^{\star}) = -\bigtriangledown ^Tf(x^{\star})\bigtriangledown f(x^{\star}) < 0
\ &\to 不符合实际（局部最小值）
\end{aligned}$
​

2. 若f(x) 在 $x^{\star}$ 处可导且取得局部最小值，且$\bigtriangledown f(x^{\star}) = 0$，则$\bigtriangledown^2f(x^{\star}) >=0 （半正定矩阵）$

证明：
$\begin{aligned}
&f(x) = f(x^{\star}) + \bigtriangledown^Tf(x^{\star})(x-x^{\star}) + \frac{1}{2}(x-x^{\star})^T\bigtriangledown^2f(x^{\star})(x-x^{\star}) + O(||x-x^{\star}||) \
&\to f(x) - f(x^{\star}) = \frac{1}{2}(x-x^{\star})^T\bigtriangledown^2f(x^{\star})(x-x^{\star}) >= 0 \
&\to(x-x^{\star})^T\bigtriangledown^2f(x^{\star})(x-x^{\star}) \
&\to \lambda^T\bigtriangledown^2f(x^{\star})\lambda >= 0 \
\end{aligned}$
​

则必要条件为: $\bigtriangledown_{x} f(x^{\star}) = 0 \quad and \quad G = \bigtriangledown^2_{x}f(^{\star}) >= 0$
​

2. 等式约束优化

$\begin{aligned}
&\min_{x\in X } \quad f(x) \
&s.t \quad g_i(x) = 0 \quad i = 1,2… \
\end{aligned}$


​

其实一般有两种做法，拉格朗日乘法和消元法，但是消元法遇到等式约束较多时比较难以处理。因此，使用拉格朗日乘法较为方便。
​

转化为拉格朗日乘法:
$\min_{x\in X} F(x,\lambda) = f(x) + \sum_i \lambda_ig_i(x)$
​

以一个具体的例子为例：
$f(x) = x_1 + x_2 \quad and \quad g(x) = x_1^2 + x_2^2 - 2$






没有加等式约束时，它的最小值时不存在的（无穷小）
而加了等式约束之后，假设当前点为$x_F$, 下一点为 $x_F + \delta x$ , 则需要有：
$f(x_F + \delta x) < f(x_F) \quad and \quad g(x_F + \delta x) = 0$


要保证:$f(x_F + \delta x) < f(x_F)$,
$f(x_F + \delta x) = f(x_F) + \bigtriangledown^T f(x_F) \delta_x + O(|\delta x)|) < f(x_F) \
\to -\bigtriangledown f(x_F) \delta_x > 0$
要保证: $g(x_F + \delta x) = 0,$要求：
$g(x_F + \delta x) = g(x_F) + \bigtriangledown^T g(x_F)\delta x = 0 \
\to \bigtriangledown^T g(x_F)\delta x = 0$


同样由几何意义可知，



即，f(x) 取得极值的地方为: 约束g(x) = 0 与 f(x) = C （等值线）相切的地方，如上图M1,M2。
此时有：

$$\bigtriangledown f(x) = \mu\bigtriangledown g(x) = 0$$


因此可以，构造 $F = f(x) + \mu g(x)$, 并要求：

$$\begin{cases}
\bigtriangledown F_x(x^{\star},\mu^{\star}) = 0 \
\bigtriangledown F_{\mu}(x^{\star},\mu^{\star}) = 0 \
\lambda^T\bigtriangledown_xx^2 L(x^{\star},\mu^{\star})\lambda >= 0 （半正定）
\end{cases}$$


另一种理解，KKT条件


构造 $\theta_p(x) = \max_{\mu} L(x,\mu) = \max_{\mu} {f(x) + \mu g(x)}$, 假设在 $x^{\star}$处取得最大值，存在两种情况：

  1. x 满足 $g(x) = 0$, 此时 $\theta_p(x) = f(x)$
  2. x 不满足 $g(x) = 0$, 此时 存在 $\theta_p(x) = +\infty$

​

而， $\min_{x} \theta_p(x) = \min_x \max_{\mu}L(x,\mu) = \min_{x} f(x)$
​

此时要求条件为：
​

$\begin{cases}
\nabla_xL(x^{\star},\mu^{\star}) = 0 \
g(x^{\star}) = 0 \
\end{cases}$

3. 不等式约束

$\begin{aligned}
&\min_{x\in X } \quad f(x) \
&s.t \quad g_i(x) \leq 0 \quad i = 1,2… \
\end{aligned}$




假设局部最小值点为 $x^{\star}$, 此时存在两种情况：

$x^{\star}$ 在约束条件之内


此时，处理方式与无约束问题一样

$x^{\star}$ 在约束条件之外

​
​

此时由几何关系可知，局部最小值位于g(x) = 0 与 f(x) = C 的切线上，由等式约束可知，此时有
​

$\nabla g(x) = -\lambda \nabla f(x) \quad and \quad \lambda > 0$
综合以上有:
​

拉格朗日乘法: $L(x,\lambda) = f(x) + \lambda g(x), \lambda >= 0$
​

$\begin{cases}
\nabla_x L(x^{\star},\lambda^{\star}) = 0 \
\lambda^{\star} \geq 0 \
\lambda^{\star}g(x^{\star}) = 0 \
g(x^{\star}) \leq 0\
\end{cases}$
以上便是KKT条件


另一个直观的解释为:


构造 $\theta_p(x) = \max_{\mu} L(x,\mu) = \max_{\mu} {f(x) + \mu g(x)}, \mu >= 0$, 假设在 $x^{\star}$处取得最大值，存在两种情况：

  1. x 满足 $g(x) = 0$, 此时 $\theta_p(x) = f(x)$
  2. x 满足 $g(x) < 0$, 此时 $\mu = 0 \to \theta_p(x) = f(x)$
  3. x 不满足 $g(x) \leq 0$, 此时 $\theta_p(x) = +\infty$


而， $\min_{x} \theta_p(x) = \min_x \max_{\mu}L(x,\mu) = \min_{x} f(x)$


此时要求条件为：


$\begin{cases}
\nabla_xL(x^{\star},\mu^{\star}) = 0 \
g(x^{\star}) \leq 0 \
\mu^{\star} >= 0\
\mu^{\star} g(x^{\star})= 0\
\end{cases}$


4. 等式和不等式约束

$\begin{aligned}
&\min_{x\in X } \quad f(x) \
&s.t \quad \begin{cases}
g_i(x) = 0 \quad i = 1,2… \
h_j(x) <= 0 \quad j = 1,2…\\end{cases}
\end{aligned}$


此时，是2，3 的综合情况，只需要构造拉格朗日乘法
$L(x,\lambda,\mu) = f(x) + \sum_i \lambda_ig_i(x) + \sum_j \mu_j h_j(x) \quad, \quad\mu_j >= 0$
若$x^{\star}$满足此条件,则有：
​

$\begin{cases}
\nabla_xL(x^{\star},\lambda^{\star},\mu^{\star}) = 0 \
g_i(x^{\star}) = 0 \
h_j(x^{\star}) \leq 0 \
\mu^{\star} >= 0\
\mu^{\star} h_j(x^{\star})= 0\
\end{cases}$
